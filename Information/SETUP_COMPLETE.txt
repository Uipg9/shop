ADDING YOUR FIRST FEATURE - EXAMPLE
================================================================================

Let's add a simple command to test everything works!

1. Open: src/main/java/com/qolmod/QOLMod.java

2. Add imports at the top:
   import net.fabricmc.fabric.api.command.v2.CommandRegistrationCallback;
   import net.minecraft.server.command.CommandManager;
   import net.minecraft.text.Text;

3. In the onInitialize() method, add:
   CommandRegistrationCallback.EVENT.register((dispatcher, registryAccess, environment) -> {
       dispatcher.register(CommandManager.literal("qol")
           .executes(context -> {
               context.getSource().sendFeedback(() -> 
                   Text.literal("QOL Mod is working!"), false);
               return 1;
           })
       );
   });

4. Build: cmd /c gradlew.bat build

5. Run: cmd /c gradlew.bat runClient
