EXAMPLE: YOUR FIRST FEATURE
================================================================================

Let's implement a simple "Tool Durability Warning" feature:

1. Create a client-side event handler:

// In QOLModClient.java
@Override
public void onInitializeClient() {
    ClientTickEvents.END_CLIENT_TICK.register(client -> {
        if (client.player != null) {
            ItemStack mainHand = client.player.getMainHandStack();
            if (mainHand.isDamageable()) {
                int durability = mainHand.getMaxDamage() - mainHand.getDamage();
                int maxDurability = mainHand.getMaxDamage();
                
                // Warn at 10% durability
                if (durability < maxDurability * 0.1 && durability > 0) {
                    // Show warning (implement visual/audio warning)
                    QOLMod.LOGGER.info("Tool low durability: " + durability);
                }
            }
        }
    });
}

2. Test by using a tool until low durability
3. Add visual indicator or sound
4. Add config option to enable/disable
5. Add config for durability threshold
