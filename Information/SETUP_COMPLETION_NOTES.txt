TROUBLESHOOTING
---------------

1. cmd /c gradlew.bat clean
2. Delete .gradle and build folders
3. cmd /c gradlew.bat build --refresh-dependencies

If IntelliJ doesn't show run configs:
1. File ‚Üí Close Project
2. Delete .idea folder
3. Reopen project
4. cmd /c gradlew.bat ideaSyncTask
